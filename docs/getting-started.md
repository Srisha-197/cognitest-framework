# Getting Started

Use this page as the fastest onboarding path for new users.

## Quick Start Flow

1. Install prerequisites and dependencies.
2. Create your first test using the standard structure.
3. Run locally and verify output artifacts.
4. Trigger execution through API for CI integration.
